- Logiciel de gestion de couleurs DMX -

En stage dans un studio d'enregistrement, nous avons mis en place un système de lumières en DMX, j'ai alors créé un logiciel permettant de choisir la couleur souhaitée. Le système est très basique et n'es en réalité qu'un système de séléction de couleurs donnant en résultat les valeurs de rouge, vert et bleu à utiliser dans le logiciel. Le code est opensource.

Java JavaFX
