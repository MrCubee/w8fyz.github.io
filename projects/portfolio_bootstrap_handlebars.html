- Portfolio utilisant Bootstrap et Handlebars -

Etant déjà familiarisé avec cette librairie CSS, j'ai voulu aller plus loin et j'ai utilisé ExpressJS et Handlebars. J'ai au final décidé de recommencer le projet n'étant pas un grand fan de ExpressJS ni de Bootstrap. Le code source est tout de même retrouvable sur GitHub en opensource
